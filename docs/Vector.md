# Vector

vector is a list of cordinate number `[x, y]` or `[x, y, z]`

- plus vector

`[ax, ay, az] - [bx, by, bz] = [ax - bx, ay - by, az - bz]`

result = adge from a point to b point

- minus vector

`[ax, ay, az] + [bx, by, bz] = [ax + bx, ay + by, az + bz]`

result = the new point come from (0, 0) link a and b point to triangle
